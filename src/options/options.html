Usage Tips

Free Mode: Best when you want zero API dependency for translation.
AI Service Switch: Keep the Gemini key stored, but disable AI analysis anytime.
Gemini TTS Switch: Turn Gemini TTS off to force browser SpeechSynthesis.
Fallback: If Gemini TTS is disabled or unavailable, speech uses the browser voice engine.
